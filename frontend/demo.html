✅ Spot Marker Color Fix - Visual Demonstration

Problem: Spot markers were always green, regardless of dominant player.

Solution: Markers now use the dominant player's color from dominant_player_color.

Examples:

No Dominant Player Uses neutral gray (#888) as fallback
Player 1 Dominant Color: #FCBAD3 (Pink) - player1's color
Player 2 Dominant Color: #FF8B94 (Coral) - player2's color
Player 3 Dominant Color: #4CAF50 (Green) - example player color
Player 4 Dominant Color: #2196F3 (Blue) - example player color
Player 5 Dominant Color: #FF9800 (Orange) - example player color

Technical Details:

  • ✅ Background color uses var(--dominant-color) instead of hardcoded green
  • ✅ White 2px border maintained for visibility
  • ✅ Glow effect uses dominant player color
  • ✅ Fallback to neutral gray (#888) when no dominant player exists
  • ✅ Cooldown state styles (ready/partial/cooldown) also use dominant color
